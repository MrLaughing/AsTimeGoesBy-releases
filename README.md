# AsTimeGoesBy

**时光流淌，壁纸常新。**

AsTimeGoesBy 是一款专为 E-Ink 墨水屏设备打造的壁纸应用。它不只是一个壁纸工具，更是一扇随时间呼吸的窗——让冰冷的屏幕拥有温度，让每一天都有不同的风景。

---

## ✨ 功能特性

### 🎨 壁纸设计

- **多种设计风格** — 内置丰富的壁纸模板：简约时钟、日历风格、诗词意境、格言卡片等
- **本地图片支持** — 选择相册中的照片作为壁纸背景，自定义属于你的风格
- **书籍封面壁纸** — 自动获取微信读书正在阅读的书籍封面，生成独特的阅读主题壁纸
- **花园主题壁纸** — 在壁纸上呈现静谧生长的手绘花园景观
- **实时预览** — 设计过程中的每一处修改都可以即时预览效果

### ⏰ 时间与信息

- **个性化时钟** — 支持多种时间日期格式和字体风格
- **阅读数据展示** — 在壁纸上显示当日阅读时长、阅读进度等数据
- **日历视图** — 内置日历组件，重要日期一目了然

### ☁️ 云服务

- **阿里云盘同步** — 壁纸自动同步到阿里云盘，多设备之间无缝切换
- **自动更新检查** — 启动时自动检查新版本，一键下载安装

### 🌙 墨水屏优化

- **E-Ink 专属设计** — 针对墨水屏的刷新特性优化，减少残影
- **低功耗设计** — 极致省电，不影响设备续航
- **灰度优化** — 针对墨水屏有限的灰阶进行渲染调优，画面清晰自然
- **沉浸式体验** — 全屏沉浸模式，隐藏系统导航栏

### 🔄 自动化

- **定时换壁** — 支持按时间间隔自动更换壁纸
- **开机自启** — 设备启动时自动运行
- **阅读联动** — 配合阅读应用，自动更新壁纸上的阅读数据
